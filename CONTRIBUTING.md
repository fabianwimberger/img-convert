# Contributing

## Quick start

- **Bugs:** Open an issue using the bug-report template.
- **Features:** Open an issue using the feature-request template.
- **PRs:** Fork, branch from `main`, keep the change focused, open against `main`.

## Conventions

- Prefix commits semantically (`feat:`, `fix:`, `docs:`, `ci:`, `deps:`, `refactor:`).
- One logical change per PR.
- Run `cargo fmt --all` and `cargo clippy --all-targets -- -D warnings` before pushing.
- Run `cargo test` and make sure it stays green.
